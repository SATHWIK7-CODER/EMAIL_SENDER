# Python Email Sender 📧

A Python script that sends automated emails using SMTP.

## Features
- Send personalized emails to multiple recipients
- Supports file attachments
- Error handling included

## Libraries Used
- smtplib
- email.mime

## How to Run

1. Clone this repository
2. Add your Gmail and App Password in the script
3. Run the script:

    python gmail.py

## Output
Login successful!
Email sent to Alice (alice@example.com)
Email sent to Bob (bob@example.com)
Done! 2 email(s) sent successfully.
